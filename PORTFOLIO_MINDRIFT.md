# 🛰️ TECHNICAL PORTFOLIO: UNIVERSAL DATA HARVESTING UNIT

## **Project Overview**
Developed a **Universal Master Scraper (Python)** designed to navigate complex site hierarchies across heterogeneous platforms. My role involved "piloting" high-level automation to solve non-trivial data extraction problems.

## **Case Study 1: "Ultimate" E-Commerce Data Harvesting (DNM.EG)**
*   **Challenge:** Consolidating every single data point from a distributed e-commerce site into a unified master file.
*   **Solution:** Built a hybrid extraction engine that merges Shopify API responses with deep HTML-parsed data (CSS selectors, regex extraction).
*   **Result:** Created the "Ultimate Site Dataset" – a comprehensive master JSON containing 28+ product variants, 335+ high-res images, full description blocks, navigational schemas, and legally-binding policy texts.

## **Case Study 2: Governmental Audit (AARO.MIL)**
*   **Challenge:** Navigating a secure .mil domain with deep hierarchical folders and high security.
*   **Solution:** Engineered a multi-level crawler with **stealth headers** and **incremental saving** to handle JS-rendered links and complex declassified reports.
*   **Result:** 35+ verified datasets (including material analysis and historical papers), historical research PDFs, and declassified imagery extracted for analysis.

## **Technical Stack (The "Vibecode" Workflow)**
*   **Language:** Python 3.11+.
*   **Frameworks:** Scrapling, BeautifulSoup, Requests.
*   **Validation:** Incremental JSON-to-CSV normalization logic.
*   **AI Collaboration:** Utilized AI Agents to troubleshoot site architecture changes and generate custom parsing logic in real-time.

# 🕷️ مشروع ماكينة الكشط الشاملة (Universal Scraping Project)

تم بناء نظام متكامل لكشط ومسح البيانات من أي موقع إلكتروني في العالم، مع القدرة على فهم المنصات المختلفة واستخراج البيانات الخام (API) والبيانات المرئية (HTML).

---

## 🛠️ الأدوات البرمجية المطورة

### 1. master_scraper.py (النسخة الاحترافية)
هذا هو السكريبت "العمود الفقري" للمشروع، ويتميز بـ:
*   **اكتشاف المنصة آلياً:** يعرف لوحده لو الموقع (Shopify أو WordPress أو عادي).
*   **سحب الـ APIs:** يجيب البيانات الخام (منتجات، أسعار، مقالات) مباشرة من سكريبتات الموقع الخلفية.
*   **زحف كامل (Full Crawl):** يتبع كل الروابط الداخلية للموقع ويسحب محتوى كل صفحة.
*   **الحفظ التلقائي:** يحفظ البيانات بعد كل صفحة يتم سحبها لضمان عدم ضياع المجهود.
*   **استخراج شامل:** (نصوص، صور، فيديوهات، روابط، SEO، Schema، جداول).

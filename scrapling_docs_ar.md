# Scrapling: الدليل والتوثيق الكامل باللغة العربية

**Scrapling** هو إطار عمل حديث وعالي الأداء لكشط وزحف الويب (Web Scraping & Crawling) بلغة بايثون. مصمم للتعامل مع كل شيء من طلبات HTTP البسيطة إلى عمليات الزحف المعقدة والمحمية بأنظمة مكافحة البوتات، مع ميزات فريدة مثل "المحلل التكيفي" والتكامل مع الذكاء الاصطناعي.

---

## 🚀 1. نظرة عامة (Overview)
يتميز Scrapling بالجمع بين السرعة، التخفي، و "الذكاء" في تحديد العناصر.
- **تخفي فائق (Stealth):** تجاوز مدمج لـ Cloudflare Turnstile، ومحاكاة دقيقة للمتصفحات.
- **القدرة على التكيف (Adaptability):** يمكنه العثور على العناصر حتى لو تغير تصميم الموقع.
- **أداء عالي:** مُحسّن للسرعة واستهلاك منخفض للذاكرة.
- **جاهز للذكاء الاصطناعي:** يدعم خادم MCP لتزويد النماذج الذكية (مثل Claude/Cursor) بالبيانات فوراً.

---

## 🛠 2. التثبيت (Installation)
يتطلب بايثون **3.10** فما فوق.

```bash
# للتثبيت الكامل (يُنصح به):
pip install "scrapling[all]"

# ثم تثبيت المتصفحات اللازمة:
scrapling install
```

## 🌐 3. أدوات الجلب (Fetchers)
المسؤول عن جلب محتوى الموقع، وهناك 3 أنواع أساسية:

### أ. `Fetcher` (طلب HTTP عادي)
سريع وخفيف، مثالي للمواقع البسيطة غير المحمية. يدعم HTTP/3 ومحاكاة بصمات المتصفح.

### ب. `StealthyFetcher` (لتخطي الحماية)
مصمم للعمل "تحت الرادار". يتجاوز Cloudflare وأنظمة مكافحة البوتات تلقائياً ويدير الجلسات (Sessions).

### ج. `DynamicFetcher` (متصفح كامل)
يستخدم Playwright/Chrome للمواقع التي تعتمد على Javascript بشكل ثقيل (مثل المواقع التي تطلب التمرير للأسفل لتحميل البيانات).

## 🔍 4. التحليل والاختيار (Parsing & Selection)
محرك اختيار قوي يدعم عدة طرق:
```python
# استخدام CSS Selectors
page.css(".product-title")

# ميزة التحليل التكيفي (السحرية) 🧠
# يمكنك حفظ "DNA" العنصر وعند تغير الموقع لاحقاً، سيتمكن Scrapling من إيجاده تلقائياً
product = page.css(".price", auto_save=True) # حفظ
product = page.css(".price", adaptive=True)  # استرجاع حتى لو تغير الكود
```

## 🕸 5. إطار عمل العناكب (Spiders Framework)
لبناء زواحف كبيرة الحجم مع إدارة الجلسات بشكل مستمر.
- **الإيقاف والاستئناف (Pause & Resume):** يمكنك إغلاق الزاحف وإكماله لاحقاً من حيث توقف.
- **البث (Streaming):** معالجة البيانات بمجرد وصولها في الوقت الفعلي.

---

## 🐚 6. سطر الأوامر (CLI)
يوفر Scrapling "صدفة" (Shell) تفاعلية لتجربة الأكواد بسرعة:
```bash
scrapling shell https://example.com
```
داخل الـ Shell يمكنك تجربة الأوامر مثل `page.css()` ورؤية النتائج فوراً.

## 🤖 7. التكامل مع الذكاء الاصطناعي (AI/MCP)
يحتوي على خادم **MCP** مدمج، مما يسمح لأدوات مثل Claude و Cursor باستخدام Scrapling كمساعد آلي لسحب البيانات وتحليلها بدقة وتكلفة أقل.
